# Versiones

Historial de versiones del proyecto SistemSo.

## Versionado Semántico

El proyecto sigue [Semantic Versioning 2.0.0](https://semver.org/):

- **MAJOR.MINOR.PATCH** (ej: 1.5.2)
- **MAJOR**: Cambios incompatibles con versiones anteriores
- **MINOR**: Nueva funcionalidad compatible
- **PATCH**: Correcciones de bugs compatibles

## Proceso de Liberación

Las versiones se generan automáticamente cuando se hace merge a las ramas principales:

1. El sistema analiza los commits desde la última versión
2. Determina el tipo de incremento según los cambios
3. Genera un nuevo tag con la versión
4. Actualiza el CHANGELOG
5. Crea documentación específica de la versión
